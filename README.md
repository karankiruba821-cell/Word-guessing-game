# 🎮 Python Logic Games

This repository contains simple console-based Python games created to improve logical thinking and programming skills.

## 🧠 About the Project

These games are built using basic Python concepts such as:
- Loops
- Conditional statements
- Functions
- Random module
- User input handling

The purpose of this project is to practice logic building and strengthen problem-solving ability.

## 🎯 Games Included

1. Secret Number Game
2. Word Scramble Game
3. Hangman Game (Basic Version)

More games will be added soon!

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run any game file using:

python filename.py
